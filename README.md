
# VISI SECURITY

Във всяка система има уязвимости, предствляващи значителна заплаха за потребителите. Начин за отстраняването им е чрез ъпдейтване на системата. Нашият проект представлява Python приложение, което позволява на потребителите бързо и лесно да поддържат софтуера си ъпдейтнат.
